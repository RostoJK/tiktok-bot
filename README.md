\# TikTok Telegram Bot (Python + Aiogram 3 + Webhook + Render)



\## 1. Установи переменные окружения:

\- BOT\_TOKEN — токен от BotFather

\- WEBHOOK\_URL — https://your-app.onrender.com/webhook



\## 2. Деплой на Render:

\- New Web Service

\- Build: pip install -r requirements.txt

\- Start: python main.py



\## 3. Готово!

Бот принимает ссылку на TikTok и отправляет видео.
